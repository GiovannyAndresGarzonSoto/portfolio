Portafolio

Clon Manga Plus

manga-api-node-mongo manga-app-vue3-scss

Este proyecto es un Clon de la pagina MangaPlus la cual sirve para leer mangas de manera online. Las tecnologías utilizadas en el proyecto fueron:
Frontend: Vue.js 3 con CompositionAPI, TypeScript, Axios y SCSS.
Backend: Node.js 18, TypeScript, Express, MongoDB, mongoose y Cloudinary para el manejo y almacenamiento de imagenes.
El despliegue del backend fue realizado en Railway, el cual proporciona una capa gratuita muy versátil. Para el frontend, utilicé Netlify, el cual de la misma manera incluye una capa gratuita.
